# API (бэкап)

Роуты `/api/coins` и `/api/coins/[id]` перенесены сюда, потому что при деплое только папки **out** (статический экспорт) API на сервере не выполняется.

Сейчас каталог и страницы монет берут данные из статических JSON: **/data/coins.json** и **/data/coins/[id].json** (они попадают в out при `npm run build` после скрипта экспорта из БД).

Если позже будете поднимать сайт с Node (`next start`), верните папку обратно: `mv api-backup app/api`.
