# Рекомендации по SEO (органический рост без рекламы)

Записано для последующей реализации. Ничего не менять без согласования.

---

## Приоритет 1 — Максимальный эффект

### Уникальные title и description для каждой монеты
- Сейчас у всех страниц монет один общий title/description.
- Нужно: `generateMetadata` в `app/coins/[id]/page.tsx` — брать из данных монеты: название, металл, год, номинал.
- Эффект: длинный хвост запросов («монета Зайка Сочи», «3 рубля серебро 1992») начнёт приводить на конкретные страницы; каждая страница — отдельная точка входа из поиска.

### Sitemap + robots.txt
- Sitemap: главная, каталог, все `/coins/[id]/`.
- robots.txt: указать путь к sitemap.
- Эффект: быстрее и полнее индексация, больше страниц в выдаче.

### Open Graph (превью в соцсетях и мессенджерах)
- Для главной и страницы монеты: og:title, og:description, og:image, og:url.
- Эффект: при расшаривании ссылки выглядят аккуратно, больше переходов из «сарафанного радио».

---

## Приоритет 2 — По возможности

- Отдельные metadata для `/catalog/` (title, description).
- JSON-LD на странице монеты (Product / коллекционный предмет) — для расширенных сниппетов.
- Canonical URL на ключевых страницах.
- BreadcrumbList в разметке (каталог → монета).

---

## Что меньше влияет на рост без рекламы

- Борьба за общие запросы («купить монеты», «нумизматика») без бюджета — конкуренция высокая.
- Сложная техническая SEO-оптимизация — вторична по сравнению с уникальными мета и sitemap.

---

## Порядок внедрения

1. `generateMetadata` для страницы монеты.
2. Sitemap + robots.txt.
3. Open Graph (главная + страница монеты).
4. Остальное по мере сил.
