# Варианты: русское название + поиск на английском

## Задача

- **В БД хранить на русском** (отображаем пользователю)
- **Поиск работал и на русском, и на английском** (например, «American Silver Eagle» находил «Американский серебряный орёл 2021»)

---

## Вариант 1: Столбец `title_en`

**Схема:** добавить `title_en VARCHAR(500) NULL` в `coins`.

| Поле      | Назначение                              |
|-----------|-----------------------------------------|
| `title`   | Русское название (для отображения)     |
| `title_en`| Английское (для поиска, SEO, импорта)   |

**Поиск:** в `coinMatchesSearch` проверять и `title`, и `title_en`:
```ts
const fields = [
  coin.title,
  coin.titleEn,   // новое
  coin.seriesName,
  ...
];
```

**Плюсы:**
- Простая схема
- Явное разделение языков
- Для иностранных монет английское название обычно уже есть (ucoin, uCoin)

**Минусы:**
- Нужно заполнять при импорте
- Для российских монет — либо NULL, либо переводить вручную

---

## Вариант 2: Объединённое поле `search_text`

**Схема:** `search_text TEXT` — конкатенация «title | title_en | series | …».

**Плюсы:** один столбец для поиска.

**Минусы:** дублирование, сложнее поддерживать.

---

## Вариант 3: Таблица синонимов `coin_search_terms`

**Схема:** `coin_search_terms(coin_id, term)` — множество поисковых вариантов.

**Плюсы:** гибкость (любые языки и синонимы).

**Минусы:** лишняя таблица и JOIN при поиске, сложнее импорт.

---

## Рекомендация: Вариант 1 (`title_en`)

1. **Просто:** один столбец, без новых сущностей
2. **Удобно для импорта:** ASE и другие иностранные серии приходят на английском
3. **Правило заполнения:**
   - Иностранные монеты: `title` = русский перевод, `title_en` = оригинал (American Silver Eagle)
   - Российские монеты: `title` = русское название, `title_en` = NULL или перевод при наличии

4. **Поиск:** добавляем `title_en` в список полей в `coinMatchesSearch` и `portfolioRowMatchesSearch`

---

## Пример для ASE

| title (RU)                  | title_en                    |
|-----------------------------|-----------------------------|
| Американский серебряный орёл 2021 | American Silver Eagle 2021  |
| Американский серебряный орёл 2021 (S) | American Silver Eagle 2021 (S) |

Поиск «American» или «Silver Eagle» будет находить монету по `title_en`.
