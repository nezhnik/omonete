## Рекомендации

1. **Источник данных GovMint** опирается на U.S. Mint отчёты — это более актуальный и авторитетный источник, чем ucoin.
2. **Крупные расхождения** (2020 BU, 2016–2017 BU, 2014 BU, 2012 BU) — стоит сверить с [U.S. Mint reports](https://www.usmint.gov/) и при необходимости обновить наши значения.
3. **Proof 2020, 2018** — наши значения выше GovMint; возможна разная методика учёта (продажи vs выпуск).
4. **2022–2026** — добавить в `walking-liberty-ucoin.json` по данным GovMint / U.S. Mint.
