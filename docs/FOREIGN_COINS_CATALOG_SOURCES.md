# Откуда берутся каталоги по иностранным монетам

Вопрос: как у сайтов вроде Numista и Colnect оказываются сотни тысяч монет — неужели всё вручную? Краткий ответ: **часть — действительно сообщество и годы ручного ввода, часть — коммерческие каталоги и официальные данные**. Единого бесплатного «каталога мира в одном файле» нет, но источники есть.

---

## 1. Сообщество и ручное пополнение (основа крупных сайтов)

### Numista
- Каталог **полностью ведут пользователи**: добавление и правки через формы, проверка модераторами (referees).
- Обязательные поля при добавлении: название, эмитент, тип, описание/надписи, изображения (или описание), дата(ы), номинал/валюта, хотя бы один каталожный номер (Krause, другой).
- Сотни тысяч типов монет и банкнот накопились за счёт **множества участников и лет работы**, а не одного импорта.

### Colnect
- Каталоги тоже **создаются сообществом**, правки проверяют волонтёры-редакторы.
- Есть экспорт CSV (до 100 записей бесплатно, до 10 000 — Premium) и **API** (по заявке, бесплатно для некоммерческого использования). То есть данные можно получать машинами, но наполнение изначально — людское.

**Вывод:** большие объёмы на Numista/Colnect — это в основном долгосрочное краудсорсинговое наполнение, а не один чей-то «украденный» каталог. Ручная работа распределена по тысячам пользователей и годам.

---

## 2. Коммерческие каталоги и базы (готовые данные)

### Krause / Standard Catalog of World Coins
- Печатные тома по странам и векам, номера KM (Krause–Mishler) — стандарт идентификации.
- **Цифра:** NumisMaster (numismaster.com) — платные поисковые PDF по странам (порядка $8–13 за страну/период). Полной «базы в Excel» в открытом доступе нет, лицензии — через правообладателей.

### CoinsCatalogs.com — World Coins Database
- **Платно:** база в **Excel** по томам (по первой букве страны: A, B, C…). Один том (например, страны на «A») — десятки тысяч монет (например, 14 495 в томе 1), цена порядка $15.99 за том, 13 томов.
- **Бесплатно:** «World Coins Catalogue 2023» — **172 734 монеты** (от 700 до н.э. до 2023), в виде иллюстрированного каталога (скорее PDF/печать), не Excel.

Здесь уже «каталог в файлах» есть, но платный (Excel) или бесплатный в неудобном для парсинга виде (большой PDF).

### World Coin Database (worldcoindatabase.com)
- Независимый каталог, ориентируется на стандарт Krause, фокус на **современную машинную чеканку**. Свой структурированный каталог, без массового бесплатного вывода в CSV/Excel в поиске.

## 3. Официальные данные монетных дворов и госструктур

- **США (US Mint):**  
  - Официальный каталог: catalog.usmint.gov.  
  - **Данные по тиражам/продажам буллона** (American Eagle и др.) — на [data.gov](https://catalog.data.gov/dataset/bullion-sales-and-mintagefigures), обновляются, можно использовать для проверки/импорта тиражей.

- **Австралия (Royal Australian Mint):**  
  - Отчёты и таблицы по производству монет — в PDF годовых отчётов (например, 2020–21). Отдельного общедоступного CSV/Excel по всем монетам нет, но данные по тиражам и типам можно извлекать из отчётов.

- **Perth Mint:**  
  - Нет готового каталога в Excel/CSV; есть магазин с пагинацией и PDF-каталоги для коллекционеров. Подход у нас описан в `PERTH_MINT_IMAGE_URLS.md` и `COLNECT_PERTH_MINT_CATALOG.md` (список можно взять с Colnect, картинки — с сайта Perth Mint).

Итого: **ни один монетный двор не отдаёт «весь мир» в одном файле**, но по отдельным странам/дворам бывают каталоги, отчёты или data.gov-подобные наборы.

## 4. Открытые и академические источники (RDF/JSON/API)

- **Nomisma.org**  
  - Открытые данные по нумизматическим концепциям (типы, эмитенты, метаданные). Форматы: RDF/XML, JSON-LD, Turtle. Лицензия CC-BY.  
  - Охват: и античность, и часть **современной нумизматики** (по странам есть разделы «Modern … Numismatics»). Подходит для связки типов монет с внешними каталогами и для структурированного импорта там, где данные есть.

- **American Numismatic Society (ANS)**  
  - API (NUDS/XML, RDF, JSON-LD и др.) — в первую очередь для научных/музейных данных (античные и исторические монеты, клады). Не каталог «всех мировых монет», а специализированные наборы.

- **Wikidata**  
  - Есть свойства «Numista coin ID» и «Numista mint ID» — часть статей о монетах привязана к Numista. Можно строить списки по странам/валютам (например, «Coins of the pound sterling») и использовать Numista как источник деталей через API/идентификаторы.

Эти источники дают не «один большой каталог иностранных монет в таблице», а **структурированные данные по тем областям, которые ими уже покрыты** (античность, отдельные страны, связка с Numista).

---

## 5. Программы и импорт из крупных каталогов

- **OpenNumismat**  
  - Бесплатное ПО для коллекций. Есть **готовые каталоги** по отдельным странам/темам: США (оборот), евро, Россия (памятные и буллон 1992–2013), Украина, Польша, Болгария, Испания и др. Формат — свои файлы каталогов.  
  - Программа умеет **импортировать из Colnect, Numista, uCoin.net** и др. То есть часть «каталога» пользователи получают уже с этих сайтов, а не из одного первичного файла.

- **Numista API / Colnect API**  
  - Позволяют получать списки и поля монет программно. Данные изначально собраны сообществом (и, возможно, частично из печатных/коммерческих каталогов при раннем наполнении), а не «у кого-то одним файлом».

---

## Сводка: есть ли «каталог по иностранным монетам»?

| Тип источника              | Есть ли каталог? | Формат / ограничения |
|---------------------------|------------------|-----------------------|
| Numista / Colnect         | Да, огромный     | Веб + API/экспорт; наполнение — сообщество |
| Krause / NumisMaster      | Да               | Платные PDF по странам, лицензии |
| CoinsCatalogs.com         | Да               | Платной Excel по томам; бесплатно — большой PDF 2023 (172k монет) |
| US Mint / data.gov        | Частично         | Тираж/продажи буллона, не полный каталог мира |
| Perth Mint / ЦБ РФ        | Свой каталог     | Сайт, PDF; у ЦБ — Excel, у Perth — нет |
| Nomisma / ANS / Wikidata  | Частично         | RDF, JSON, API; не «весь мир», а выборочные наборы |
| OpenNumismat              | Готовые каталоги | По странам/темам; импорт из Numista/Colnect/uCoin |

**Итог:** массу монет сайты действительно «добывали» не из одного секретного каталога, а за счёт **сообщества (вручную, годами)** и частично — из **коммерческих каталогов (Krause, платные базы)** и **официальных данных** там, где они есть. Готового бесплатного «все иностранные монеты в одном Excel» в открытом доступе нет; для нашей задачи разумно комбинировать: Colnect/Numista (списки/API), платные Excel при необходимости, официальные данные по тиражам и, где нужно, парсинг сайтов дворов (как у Perth Mint).
