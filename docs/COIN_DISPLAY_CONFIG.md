# Первая картинка у монет

В корне проекта файл **coin-display-config.json**:

```json
{
  "firstImage": "obverse"
}
```

- **obverse** — первой показывается аверс (уникальное изображение). В каталоге и в деталях монеты.
- **reverse** — первой показывается реверс (орёл с датой и номиналом у российских монет).

После изменения пересобери: `npm run build`.

В БД ничего менять не нужно: столбцы `image_obverse` и `image_reverse` остаются как есть, меняется только порядок показа на сайте.
