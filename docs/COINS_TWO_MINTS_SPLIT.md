# Монеты с двумя монетными дворами: разделение на отдельные страницы

## Что нашли в интернете

- **ЦБ РФ:** в каталоге одна позиция на тип монеты; в описании указано «Чеканка: Московский и Санкт-Петербургский монетные дворы (ММД, СПМД)». Отдельных каталожных номеров для ММД и СПМД ЦБ не присваивает.
- **Различие:** на монете различается только товарный знак монетного двора (под гербом на аверсе). Дизайн и характеристики одинаковые.
- **Примеры:** Георгий Победоносец (золото 100 руб., серебро 3 руб.), Зайка, Белый мишка, Соболь, Червонец 1992 — чеканились на двух дворах.

Чтобы не вводить в заблуждение («у монеты два двора»), на сайте делаем **отдельную страницу под каждый двор**: одна запись в БД = один двор. У каждой записи свой `catalog_number` с суффиксом (например `5217-0048-24-ММД` и `5217-0048-24-СПМД`), название вида «Георгий Победоносец (ММД)» / «Георгий Победоносец (СПМД)».

## Реализация

- Скрипт **`scripts/split-two-mints-to-separate-coins.js`** находит все монеты, у которых `mint_short` содержит запятую (например `"ММД, СПМД"`), и:
  1. Оставляет одну запись за «первый» двор (обновляет `mint`, `mint_short`, `catalog_number` = база + `-ММД`, `title` = «Название (ММД)»).
  2. Для каждого остального двора добавляет новую строку в `coins` (копия данных, другой двор, `catalog_number` = база + `-СПМД` и т.д.).
- Картинки общие (дизайн один, на монете различается только знак двора).
- После запуска скрипта: `npm run data:export` и `npm run build` — в каталоге и на сайте у каждой такой монеты будет своя страница с одним двором.

## Маппинг дворов

| mint_short | Полное название |
|------------|------------------|
| ММД | Московский монетный двор |
| СПМД | Санкт-Петербургский монетный двор |
| ЛМД | Ленинградский монетный двор |
