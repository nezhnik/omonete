# Colnect как каталог для Perth Mint (и других монет)

Colnect содержит большой каталог монет, в том числе **отдельный раздел по Perth Mint**. Это может помочь нам получать список монет, названия и метаданные без полного парсинга сайта Perth Mint.

---

## Что есть на Colnect по Perth Mint

- **Серия по монетному двору:**  
  [Coins › Series › Mint: Perth Mint, Perth, Australia](https://colnect.com/en/coins/series/mint/5-Perth_Mint_Perth_Australia)

- **Фильтр «Numismatic Product» + Perth Mint:**  
  [List › Distribution: Numismatic Product, Mint: Perth Mint](https://colnect.com/en/coins/list/distribution/3-Numismatic_Product/mint/5-Perth_Mint_Perth_Australia/weight/22)  
  Около **319+ позиций** (1970 – н.в.).

- **Буллон (Bullion Product), Perth Mint:**  
  Около **20 позиций** (1986 – н.в.).

- Дополнительно: proof sets, mint sets, стандартный чекан Австралии и т.д.

На страницах монет есть: название, номинал, год, серия, металл, вес, тираж (если внесён), изображения (часто от сообщества). **SKU и точные URL картинок Perth Mint на Colnect нет** — их по-прежнему нужно брать с [perthmint.com](https://www.perthmint.com) (см. `docs/PERTH_MINT_IMAGE_URLS.md`). Colnect даёт **список монет и метаданные** для сверки и для того, чтобы знать, что искать на официальном сайте.

---

## Как использовать каталог Colnect

### 1. Ручной экспорт (CSV)

- Открыть список монет (например, Numismatic Product + Perth Mint).
- Нажать **Export** на странице списка.
- **Бесплатный аккаунт:** до 100 позиций в одном экспорте.
- **Premium:** до 10 000 позиций.
- Формат: CSV (UTF-8), можно включать/исключать минорные варианты.

Удобно для небольшой выгрузки и проверки полей (название, год, номинал, вес, тираж).

### 2. Colnect API (CAPI)

- **Документация:** [Colnect API (Help)](https://colnect.com/en/help/collecting/colnect_api)
- **Базовый URL:** `https://api.colnect.net`
- **Доступ:** по ключу. Ключ выдаётся по заявке, для некоммерческого использования — бесплатно.
- **Аутентификация:** заголовки `Capi-Timestamp` и `Capi-Hash`.
- Ответы — структурированные (в т.ч. JSON), можно автоматизировать получение списков монет по стране, монетному двору, типу продукта и т.д.

Имеет смысл, если планируем регулярно подтягивать списки Perth Mint (или других дворов) и сверять их с нашим каталогом.

### 3. Парсинг страниц Colnect

- На сайте включена защита от ботов (Anubis), поэтому массовый парсинг страниц списков/карточек может быть сложным или нестабильным.
- Для разовых выгрузок проще использовать **экспорт в CSV** или **API**, если получим ключ.

## Пример страницы монеты на Colnect

Ссылка из запроса (American Silver Eagle, 250th Anniversary, Star privy):  
[https://colnect.com/ru/coins/coin/207537-1_Dollar_American_Eagle_-_250th_Anniversary_...](https://colnect.com/ru/coins/coin/207537-1_Dollar_American_Eagle_-_250th_Anniversary_United_States_-_Star_privy-F01_-_Американские_Серебряные_Орлы-Соединенные_Штаты_Америки)

На такой странице обычно есть:
- название (в т.ч. на русском, если есть),
- страна, номинал, год,
- серия (например, «Американские Серебряные Орлы»),
- металл, вес, тираж,
- изображения (аверс/реверс),
- каталожный номер/ID Colnect (в URL — `207537`).

Для **Perth Mint** страницы монет дают названия и параметры; по ним можно искать соответствие на perthmint.com (по названию/году/номиналу) и затем запускать наш скрипт `fetch-perth-mint-coin.js` по URL страницы товара.

## Итог: как Colnect помогает с Perth Mint

| Задача | Источник | Примечание |
|--------|----------|------------|
| Список монет Perth Mint (названия, годы, номинал, вес) | Colnect (экспорт CSV или API) | Удобно для списка и сверки |
| Тираж | Colnect (если заполнен) или Perth Mint / PDF | Colnect не всегда полный |
| Изображения в высоком качестве, SKU, URL картинок | Только Perth Mint | См. `PERTH_MINT_IMAGE_URLS.md` и `fetch-perth-mint-coin.js` |

**Рекомендация:** использовать Colnect как **каталог-справочник** для получения списка монет Perth Mint (и при необходимости других дворов). По этому списку определять, какие монеты нам нужны, и для каждой при необходимости открывать страницу товара на perthmint.com и запускать существующий пайплайн (скрипт + сохранение картинок и данных).

Если решим автоматизировать выгрузку списков — оформить заявку на ключ Colnect API и описать в этом документе шаги и примеры запросов.
