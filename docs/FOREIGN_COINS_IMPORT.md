# Импорт иностранных монет из ucoin.net

## Тестовый файл

**Файл:** `data/foreign-coins-test.csv`  
**Формат:** TSV (табуляция), кодировка UTF-8  
**Содержимое:** 5 тестовых монет с колонками как у российских монет в БД

## Ограничения

- **Монетные дворы:** только из нашей системы (`lib/mint-articles.ts`):
  - Монетный двор США, Королевский монетный двор Великобритании, Королевский канадский монетный двор, Австрийский монетный двор, Южноафриканский монетный двор и др.
- **Металлы:** только Серебро, Золото, Платина, Палладий, Медь (как в `lib/coinApiShape.ts` / `scripts/export-coins-to-json.js`)

## Источник данных

**Сайт:** https://ru.ucoin.net/catalog/  
**Фильтры:** country (страна), period (период), type (тип), composition (металл)

Публичного API у ucoin.net нет — данные нужно собирать парсингом страниц. Перед массовым импортом:
1. Проверить правила использования сайта
2. Реализовать парсер с фильтром по нашим mints и metals
3. Сопоставить названия дворов ucoin с нашими (mints.name)

## Парсинг ucoin.net (осторожный режим)

**Скрипты:**
- `node scripts/ucoin-fetch-page.js [url]` — сохранить HTML одной страницы в `data/ucoin-page-sample.html` (для отладки)
- `node scripts/parse-ucoin-walking-liberty.js` — парсинг серии Walking Liberty в `data/walking-liberty-ucoin.json`
- `node scripts/parse-ucoin-walking-liberty.js --dry-run` — только показать URL, без запросов

**Защита от блокировки:**
- Задержка 3–6 сек между запросами (случайная)
- User-Agent и заголовки как у браузера
- Вывод только в файл, в БД не пишем

**Важно:** ucoin может возвращать 403 при запросах не из браузера (Cloudflare и др.). Если так — сохраните страницу вручную (Ctrl+S в браузере) как `data/ucoin-page-sample.html` и доработайте парсер по этой структуре. Либо запускайте скрипты локально (домашний IP реже блокируется).

**Валидация:** данные проверяются по известным характеристикам серии (год 1916–1947, вес ~12,5 г, серебро). Монеты с расхождениями помечаются для ручной проверки.

**Фото:** планируется подбирать с официальных сайтов монетных дворов (usmint.gov и т.д.).
